FinanceOS API Documentation

Self-documenting interactive API schema for the Finance Management System.

Authentication

POST /api/auth/register New account creation
POST /api/auth/login Authenticate & return JWT
GET /api/auth/me Required Get profile info

Financial Records

GET /api/records Required List with pagination/filters
POST /api/records Analyst+ Create a record
PUT /api/records/:id Analyst+ Update a record
DELETE /api/records/:id Admin Soft-delete record

Analytics & Dashboard

GET /api/dashboard/summary Required Financial summary cards
GET /api/dashboard/trends Analyst+ Monthly trends data
GET /api/dashboard/by-category Required Doughnut chart totals
